# 1️⃣ 개념 정리 

## 10. 데이터 탐색과 시각화

```
✅ 학습 목표 :
* EDA의 목적을 설명할 수 있다.
* 주어진 데이터셋에서 이상치, 누락값, 분포 등을 식별하고 EDA 결과를 바탕으로 데이터셋의 특징을 해석할 수 있다.
* 공분산과 상관계수를 활용하여 두 변수 간의 관계를 해석할 수 있다.
* 적절한 시각화 기법을 선택하여 데이터의 특성을 효과적으로 전달할 수 있다.
```

## 분포 시각화

분포 시각화는 연속형(양적)과 명목(질적) 척도인지에 따라 구분해서 그린다.

양적(연속) : 막대그래프, 선 그래프 , 히스토그램 

* 히스토그램 : 겹치지 않는 변수의 구간(bin)을 동일하게 나눠서 구간별 도수를 막대그래프로 표현한 것

구간: bin

구간의 높이 : 밀도(density) = 가로축의 단위 구간에 속한 값의 비율을 나타냄

구간이 너무 많으면 보기 어렵고 너무 적으면 정보의 손실이 크기 때문에 시각화 이점이 사라진다

질적(명목) : 파이차트나 도넛차트를 사용

요소가 복잡한 질적 척도 : 트리맵 차트

와플 차트 : 와플 처럼 일정한 네모 조각들로 분포 표현

* 트리맵 차트 처럼 위계 구조 표현 x

## 관계 시각화

산점도 : 점들의 분포와 추세를 통해 변수 간의 관계 파악

* 극단치를 제거하고 그리는 것이 바람직

=> 두 개의 변수 간의 관계만 표현할 수 있다는 단점

버블 차트 : 여러개의 변수 관계를 표현 가능
~~~~
주의할점

1. 요소를 늘리면 한 번에 제공되는 정보가 너무 많아져 차트를 해석하는 것이 어려워진다.

2. 원의 면적을 함께 봐야한다. 이 때 원의 지름이 아닌 면적을 통해 크기를 판단하므로 지름이 두 배가 크면 실제 크기는 네 배가 큰 것이다.
~~~~

## 공간 시각화
위치 정보인 위도와 경도 데이터를 지도에 매핑하여 시각적 표현

거시적에서 미시적으로 진행되는 분석 방향과 같이 스토리라인을 잡고 시각화 적용하는 것이 좋다

도트맵 : 지리적 위치에 동일한 크기의 작은 점을 찍어서  해당 지역의 데이터 분포나 패턴을 표현하는 기법

버블맵 : 버블 차트를 지도에 그대로 옮겨 둔 것

-비율을 비교하는 것이 효과적

코로플레스맵(단계 구분도) : 데이터 값의 크기에 따라 색상의 음영을 달리하여 해단 지역에 대한 값을 시각화하는 기법

>- 명도 , 투명도, 채도 다양
>- 정확한 수치 인지 비교가 어려움

커넥션맵(링크맵) :
 1. 지도에찍힌 점들을 곡선 또는 직선으로 연결하여 지리적 관계 표현
2. 연속적 연결을 통해 지도에 경로 표현 

ex) 플로우맵 , 카토그램

## 박스플롯

하나의 그림으로 양적 척도 데이터의 분포 및 편향성, 평균과 중앙값 등 가양한 수치를 보기 쉽게 정리해줌

해석 시 유의사항 : 데이터 분포도를 함께 고려

최소

제1사분위에서 1.5 IQR을 뺀 위치

25%의 위치(작은 값)

제2사분위(Q2)

50%의 위치(중앙값을 의미)

제3사분위(Q3) 

75%의 위치(큰 값)

최대: 제3사분위에서 1.5 IQR을 더한 위치
